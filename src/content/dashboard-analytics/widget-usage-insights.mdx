# Widget Usage Insights

<Callout type="info">
Your widget doesn’t just sit idle — it collects valuable data on how users engage with accessibility tools. This page shows you how to interpret that data.
</Callout>

## What You Can Learn

WelcomingWeb tracks anonymized interaction data to help you understand how users engage with the widget across devices, locations, and preferences.

<Steps>
### Active Users
See how many unique users interacted with the widget in the last 7, 14, or 30 days.

### Top Features Used
Discover which accessibility features are most frequently enabled — like contrast toggles, text enlargement, or screen readers.

### Hourly Activity
View a usage heatmap showing when users most commonly activate the widget during the day.

### Device & OS Distribution
Track usage by desktop, mobile, tablet, and operating system (Windows, iOS, Android, etc.).

### Top Languages
Monitor what languages are most used when translation or locale preference is active.

### Screen Resolutions & Viewports
Learn which screen sizes your users are accessing the widget from, helping optimize UI decisions.

### Geographic Insights
See which countries and timezones your accessibility tools are being used from.

</Steps>

---

## How This Helps

- Identify peak usage times to optimize performance
- Understand which features matter most to your audience
- Align widget defaults with user preferences (like default language or mode)
- Justify accessibility investments with real usage data
