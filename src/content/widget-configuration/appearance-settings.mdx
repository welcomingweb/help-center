## Button Color Mode

**SOLID** for one consistent color, or **GRADIENT** for a smooth two-color transition.

## Primary & Secondary Colors

- **Primary Color**  
  Your button’s main hex color.

- **Secondary Color** (Gradient only)  
  Accent color for the gradient effect.

## Gradient Direction

Define how the two colors blend:

- `TO_RIGHT` (left → right)  
- `TO_BOTTOM` (top → bottom)  
- `DIAGONAL` (e.g. top-left → bottom-right)

## Button Size

- `SMALL` – Minimal footprint  
- `MEDIUM` – Standard size  
- `LARGE` – High visibility

## Button Icon Type

Pick the symbol that best represents your site’s audience:

| Icon       | Use Case                              |
|------------|---------------------------------------|
| Default    | Standard accessibility “wheel” icon   |
| Wheelchair | Wheelchair users                      |
| Blindness  | Low-vision / screen reader users      |
| Walking    | Mobility assistance                   |
