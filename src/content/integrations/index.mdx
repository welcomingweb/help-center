## Core Integration Methods

### 1. Universal HTML Embed
Works on any website with simple copy-paste:

{`<script src="https://cdn-01.welcomingweb.com/a11y-widget.bundle.js" 
  data-widget-id="YOUR_ID"
  data-theme="auto">
</script>`}

### 2. Advanced JavaScript API
For dynamic control:

{`window.AccessibilityWidgetAPI?.enableFeature('contrast');`}
